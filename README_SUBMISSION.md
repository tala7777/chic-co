# Chic & Co.

## Project Description
A curated luxury fashion boutique designed for the modern connoisseur. Chic & Co. blends artisanal aesthetics with a cinematic user experience, offering a sanctuary for high-end fashion discovery and identity curation.

## Features
- **Style Identity Engine**: A dynamic "Aura" quiz that matches pieces to your unique style persona.
- **The Velvet Room**: A high-contrast, cinematic lookbook experience for immersive curation.
- **Unified Bag & Treasury**: A database-backed cart system that persists your selection across sessions.
- **Curator Analytics Portal**: A full administrative suite for managing the Boutique's archives and clients.
- **Soft Luxury UI**: Premium glassmorphism effects, artisanal animations, and a refined ivory-blush-ink palette.

## Technologies & Tools Used
- **Backend**: Laravel 11, PHP 8.2
- **Frontend**: Livewire 3, Alpine.js, Vanilla CSS, Bootstrap 5
- **Database**: MySQL (XAMPP Environment)
- **Icons & Display**: Font Awesome 6, Google Fonts (Playfair Display & Lato)
- **Tools**: GitHub, VS Code, Git CLI
